# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability in this repository:

1. **Do not** open a public GitHub issue with exploit details.
2. If Security Advisories are not available, open a minimal issue **without sensitive details** and ask maintainers for a private channel.

## Scope

This repository contains **example code**. Examples are provided “as-is” and may not be audited. Always review, test, and adapt code before using it in production.
